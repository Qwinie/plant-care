# 🌿 Plant Care App

Offline plantenverzorging app — geen account, geen abonnement, geen advertenties.

## Features
- 📅 Waterschema per plant met snooze
- 🌿 AI Plant Adviseur (Claude)
- ☀️ Lichtmeter via camera
- 📷 QR-code per plant
- 🦠 Plaag & ziekte log met behandeladviezen
- 🏺 Grondmengsel bijhouden
- 📄 PDF rapport exporteren
- 🌍 Volledig offline

## Privacy
