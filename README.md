# 🌿 Plant Care App

**Live app:** https://Qwinie.github.io/plant-care/

Offline plantenverzorging — geen account, geen abonnement, geen advertenties.

## Features
- 📅 Waterschema per plant met snooze
- 🌿 AI Plant Adviseur (Claude API)
- ☀️ Lichtmeter via camera
- 📷 QR-code per plant
- 🦠 Plaag & ziekte log met behandeladviezen
- 🏺 Grondmengsel bijhouden
- 📄 PDF rapport exporteren
- 🔒 Volledig offline — alle data op je eigen apparaat
